a) Uma classe em Python é um tipo especial de estrutura que serve como um molde para criar objetos. Ela define que atributos, funcionalidades e métodos os objetos vão ter. Em programação orientada a objetos, a classe ajuda a organizar o código, agrupando informações e funções relacionadas em blocos lógicos, facilitando o reaproveitamento e a manutenção.

b) Um objeto é uma instância concreta de uma classe, algo criado a partir do molde definido pela classe. No programa, cada objeto pode ter valores e comportamentos próprios, mesmo que baseados na mesma classe.

c) Atributos são as informações armazenadas dentro de um objeto, como características. Métodos são funções definidas dentro da classe, servem para executar ações relacionadas ao objeto.

d) Herança é quando uma classe pode aproveitar características e métodos de outra classe para evitar repetir o código. Uma classe filha pode herdar tudo da classe mãe e só adicionar ou modificar o que for necessário.
